INSTALAÇÃO:

git clone <repository-url>
cd geoteste
npm install

EXECUÇÃO DO PROJETO

Executar o comando 'ember s' dentro do diretório 'geoteste'
Abrir o navegador e acessar o endereço: http://localhost:4200

BIBLIOTECAS UTILIZADAS

ember-cli-sass
ember-bootstrap-sassy
ember-ajax
ember-data
ember-google-maps

PARA TESTES

ember test
ember test --server
